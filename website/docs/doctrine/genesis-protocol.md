The Genesis Protocol exposes a simple, powerful API:

```http
POST /genesis/spawn_tool          # Spawn a tool directly
```

```bash

  -H "Content-Type: application/json" \
  -d '{

    "priority": "high"
  }'
```

**Response Time:** Less than 30 seconds from request to operational tool
