# OMEGA SDK - Build Legendary Agents

## Quick Start
```bash
pip install omega-sdk
omega create-agent --name MyAgent --capabilities ["analysis", "generation"]
omega deploy --target local
```

## Agent Templates
- **Data Analyst Agent** - SQL + visualization
- **Code Review Agent** - Security + performance analysis  
- **Research Agent** - Web scraping + synthesis
- **DevOps Agent** - Infrastructure monitoring + healing
